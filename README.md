# Growthlytics 🚀

## 📊 Project Overview
**Growthlytics** is a dedicated digital marketing and analytics agency project designed to bridge the gap between creative branding and data-backed performance. This repository serves as the central hub for the agency's web presence, promotional campaigns, and SEO strategy assets.

## 🎯 Our Focus
* **Agency Branding:** Crafting modern, high-contrast, and cinematic brand identities.
* **SEO Strategy:** Implementing data-driven insights to boost organic reach.
* **Performance Marketing:** Designing high-converting vertical video campaigns and social media assets.
* **Data Analytics:** Applying business analyst methodologies to optimize client growth.

## 🛠 Tech Stack
* **Frontend:** React / Next.js
* **Styling:** Tailwind CSS / Modern CSS (Dark Navy, Light Blue, White aesthetic)
* **Video/Motion:** Cinematic motion graphics and vertical video formatting.
* **Analytics:** SEO performance tracking and data visualization dashboards.

## 🔑 Key Deliverables
* **Brand Identity:** High-resolution logos and corporate visual assets.
* **Video Campaigns:** 20-second vertical cinematic promotional videos for social media.
* **SEO & Growth:** Comprehensive content strategies and digital marketing frameworks.

## 📈 Goals
To deliver "Ultra Sharp" digital experiences that prioritize client growth through technical precision and aesthetic excellence.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
